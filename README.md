# OS  

os-simple-shell

操作說明:  
1. 把檔案(my_shell.c child.c 以及同名執行檔)放入linux環境  
2. 終端機進入到放檔案位置  
3. 終端機輸入  ./my_shell 執行程式  
4. 終端機畫面1092929_my_shell>  
5. 以老師的說明文件及範例影片測試  
  
功能:  
1. chid持續輸出，直到指令中斷   
2. 指令q中斷child程式   
3. 輸入 child & 之時 parent 不會等 child 結束  
   
增加的功能:  
1. 有部分指令可以直接在終端機使用(ex.ls'clear'exit'ps axe)
